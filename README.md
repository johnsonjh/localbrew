# Localbrew

## Overview

Creates portable _`virtualenv`_-like Homebrew environments.

## Platforms

`localbrew.sh` is regularly tested on:

- GNU/Linux
- Darwin 21.6 (macOS 10.13.6) through 22.4 (macOS 13.3)
